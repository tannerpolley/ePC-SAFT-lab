# Pure Component Saturation-Density Reference Data

This folder owns reusable pure-component saturation-density traces whose schema
is tailored to a specific validation figure or retained source table.

Files:

- `<component>/saturation_density.csv`
  - component-owned pure-component saturation-density reference rows that may
    come from paper figures, source tables, public databases, or curated
    validation inputs.
  - Source provenance belongs in row fields, not in folder names.

Generated model outputs and exact plotted-data snapshots stay with their owning
analysis figure `results/` or `output/` folders.
