# Pure Component Association AAD Reference Data

This folder owns reusable pure-component association AAD reference rows.

Use one component folder per species:

```text
association_aad/
  methanol/association_aad.csv
  1_pentanol/association_aad.csv
  1_nonanol/association_aad.csv
```

Folder and file names describe the data type and component only. Literature
source provenance belongs in row fields such as `source_reference`,
`source_method`, and `source_table`.
