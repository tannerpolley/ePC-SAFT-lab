# Goal Boards

This folder is reserved for active or intentionally durable GoalBuddy board state.

Keep it clean:

- Create a goal subfolder only for a current, broad, restart-prone, or explicitly requested GoalBuddy workflow.
- Commit only the durable `goal.md`, `state.yaml`, and concise evidence notes that are still useful after the run.
- Do not commit generated `.goalbuddy-board/` preview files.
- Remove completed or superseded goal folders after their durable decisions have been captured in current roadmaps, ADRs, tests, or user-facing docs.
