# epcsaft 0.1.2

`epcsaft` 0.1.2 is a historical release receipt for the first PyPI Trusted Publishing workflow setup.

## Historical note

This receipt corresponds to the retired `v1.5.2` GitHub release. No rebuilt package assets are attached.

## Changes

- Added a GitHub Actions PyPI publishing workflow using PyPI Trusted Publishing/OIDC.
- Added release/tag and manual-dispatch publishing paths.
- Kept wheel and sdist building inside GitHub Actions before publishing to PyPI.
- Updated user docs so PyPI could become the primary install path once publishing was live.
