# epcsaft 0.1.1

`epcsaft` 0.1.1 is a historical package-usability receipt focused on easier installation, editable source workflows, and cleaner user documentation.

## Historical note

This release was originally published under the fork-era `1.5.1` number. The project was later unforked and the release line was reset, so this receipt records that package-usability update as `0.1.1`. No rebuilt package assets are attached to this historical receipt.

## Changes

- Added PEP 660 editable-install hooks to the custom build backend.
- Kept the backend's isolated native build directory behavior for editable installs.
- Added tests for editable-hook routing.
- Updated README and Sphinx install pages with pip, GitHub tag, local source, editable, and optional Ipopt install paths.
- Reorganized the docs index so new users saw install and usage pages first, while maintainer workflows remained available under source/contributor sections.
- Removed stale user-doc language that framed pages as coding-agent-only workflows.
