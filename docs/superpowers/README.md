# Superpowers Project

This folder preserves the monorepo's compact lab context, milestone evidence,
specs, plans, and issue mirrors.

- `PROJECT_CONTEXT.md` is the compact archive boundary and completion standard.
- `milestones/` retains scientific registries and historical map indexes.
- `specs/` and `plans/` are historical monorepo records.
- `issues/` contains point-in-time issue snapshots.
- `_templates/` preserves the former local record shapes as evidence.

These records do not own live issue intake, roadmap state, or clean-repository
production authority. Historical issue links and mirrors remain provenance.
