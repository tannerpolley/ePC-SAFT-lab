# Historical M3 - EOS

This page marks the archived monorepo provider/EOS milestone. Its former issue
queue, readiness labels, and execution order are preserved in Git history and
the retained issue/plan records; they are not a current roadmap.

The working provider implementation remains under `packages/epcsaft` as the
sole transitional runtime authority for unpromoted provider slices. New
provider work belongs to the clean provider repository and moves only through
an accepted slice promotion.
