# Historical M8 - Python Toybox

This directory preserves Python-only exploratory EOS, derivative, property,
and equilibrium analyses from the monorepo program. Its former specs, plans,
and artifact tables are retained in Git history and neighboring evidence
records; they are not an active queue.

Toybox results may inform a future slice, but they do not establish provider,
equilibrium, regression, or validation capability. Any reuse must identify the
exact source, assumptions, units, domain, and independent acceptance evidence.
