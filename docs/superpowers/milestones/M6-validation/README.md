# Historical M6 - Validation

This page marks the archived monorepo validation milestone. Former open issues,
readiness labels, and plan lists are historical metadata, not active gates or
work intake.

The retained
[equilibrium evidence registry](registries/equilibrium-evidence-registry.yaml)
records literature cases, fixtures, tolerances, blockers, and evidence
maturity. It does not activate runtime routes. Future cross-package acceptance
belongs to the clean validation repository and must operate on installed,
immutable artifacts.
