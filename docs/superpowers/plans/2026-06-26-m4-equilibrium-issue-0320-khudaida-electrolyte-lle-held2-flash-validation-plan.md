# Khudaida Electrolyte LLE And HELD2 Flash Validation Plan

**Goal:** Resolve GitHub issue #320 by making the full Khudaida 2026
electrolyte LLE figure-reproduction checker pass and by adding a retained
HELD2 flash scenario gate that proves the electrolyte extension against the
already-proven HELD 1.0-style neutral base while staying on the shared native
`NlpProblem`/Ipopt exact-Hessian production path.

**Architecture:** #320 is the reopened #191 closeout blocker. #314 remains
valid as representative public-route admission evidence, but it is no longer
sufficient evidence for electrolyte LLE model performance. The resolving PR
must prove source-data reproduction and multi-scenario HELD2 flashing through
retained checkers, plots, and tests. Accepted model rows must be generated by
the pressure-transformed Helmholtz NLP route contract with route-owned bounds,
smooth variable transforms, scaling, exact objective gradients, exact
constraint Jacobians, exact Lagrangian Hessians, and Ipopt postsolve
certification. Residual-only solves, equation-only substitutes, or adapter-owned
thermodynamic equations cannot count as production evidence.

**Tech Stack:** `packages/epcsaft-equilibrium`, public
`Equilibrium(..., route="electrolyte_lle")`, native Ipopt-backed equilibrium
diagnostics through `NlpProblem`, `VariableTransform` reduced-electroneutral
coordinates, Born SSM+DS active-block exact-Hessian receipts, Khudaida
paper-validation scripts, retained CSV/SVG/PNG/PDF plot artifacts, and pytest
contracts.

## Source Audit Receipts

- GFPE doctrine requires source-backed production validation, exact first
  derivatives, exact Lagrangian Hessians, route-owned bounds/scaling/transforms,
  sparse `NlpProblem` Jacobian/Hessian contracts, Ipopt barrier constraints,
  and staged HELD evidence. Electrolyte routes add per-phase charge balance and
  projected electrochemical potentials in a charge-neutral reduced basis.
- The M4 stage plan requires all generalized routes to use the native
  `NlpProblem` route substrate with objective, gradient, constraints, Jacobian,
  Hessian, bounds, scaling, diagnostics, and exact-Hessian production gates. It
  blocks production claims when exact derivative blocks are missing or the route
  bypasses the common NLP substrate.
- The HELD2 paper transforms strong-electrolyte mole numbers into reduced
  electroneutral coordinates, compares modified electrochemical potentials, and
  extends HELD phase discovery so electroneutrality is satisfied throughout.
- The Ascani 2022 electrolyte framework formulates electrolyte phase
  equilibrium with Gibbs minimization, material constraints, per-phase charge
  constraints, transformed variables, and mean-ionic equilibrium conditions, not
  raw single-ion equality.
- The Khudaida 2026 data set supplies experimental mixed-solvent NaCl LLE
  tie-lines, pure-component parameters, dielectric constants, binary
  interaction parameters, and ePC-SAFT/eNRTL AAD comparisons. The retained
  `analyses/paper_validation/2026_khudaida` artifact contract currently states
  that public `electrolyte_lle` model rows do not yet reproduce the salted
  tie-lines.
